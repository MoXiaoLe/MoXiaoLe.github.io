webmagic-解析结果页

发表于 2017-10-21 | 分类于 小爬虫

webmagic 把成功请求响应的内容封装成一个 Page 对象，通过调用 Page 的成员方法以及成员变量即可完成对结果页的解析与抽取。

public class Page {

    private Request request;
    private Html html;
    private Json json;
    private String rawText;
    private Selectable url;
    private Map<String, List<String>> headers;


    // 需持久化的数据
    private ResultItems resultItems = new ResultItems();
    // 新发现的种子 rul
    private List<Request> targetRequests = new ArrayList();
}

重点关注 html 和 json 两个属性。如果响应内容是 json 类型就通过 Json 对象类解析，如果是 html 类型就通过 Html 对象来解析。
